# 条件判断

## 1. 整数的判断

| =    | 等于     |
| ---- | -------- |
| -lt  | 小于     |
| -gt  | 大于     |
| -ge  | 大于等于 |
| -le  | 小于等于 |
| ne   | 不等于   |

## 2. 按照文件权限进行判断

- -r：有读的权限
- -w：有写的权限
- -x：有执行的权限

## 3. 按照文件类型进行判断

- -f: 文件存在并且是一个常规文件（file）

- -e：文件存在（existence）
- -d：文件存在并是一个目录（directory）

## 4. 多条件判断

&& 表示前面一条为true才执行后面命令

||表示前面一条指令执行为false才执行后面命令
